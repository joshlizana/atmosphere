Forge created `.env.example` at the repo root as part of JDL-65 M0 scaffolding, committed as `a00452e`. The file is the single authoritative list of environment variables the compose stack will consume across every milestone; later milestones wire services to keys declared here without reshuffling the surface. ^p001

Ten logical sections group the 40-odd keys by cluster: Postgres (superuser plus per-logical-db user/password pairs for the `lakekeeper`, `prefect`, `openmetadata`, and `grafana` databases), SeaweedFS S3 (access/secret key pair, endpoint, region, three bucket names `atmosphere`/`flink`/`loki`), Redpanda (brokers and admin URL â€” no auth keys per the single-trust-domain design), Lakekeeper (URL and warehouse name â€” no service-account credentials because auth is disabled per `lakekeeper.md` Â§Auth), Grafana (admin user/password), Prefect (API URL and API key for the self-hosted server), OpenMetadata (admin email/password and JWT secret), ClickHouse (three user/password pairs for `admin`, `quarto` read-only, and `grafana` read-only), Elementary/Quarto gh-pages publisher (fine-grained PAT plus commit author name/email for `atmosphere-bot`), and Oracle (`ORACLE_DEVICE`, `ORACLE_BATCH_SIZE`, `ORACLE_FLUSH_MS` for the sentiment-inference runtime). ^p002

Design conventions honored across the file: every secret-like value placeholder is the literal `CHANGEME` so missed fields stand out obviously in logs; non-secret defaults refer to compose service names (`postgres`, `seaweedfs:8333`, `redpanda:9092`, `lakekeeper:8181`, `prefect-server:4200/api`) rather than `localhost` because every client resolves peers over the compose bridge network; the format is POSIX `KEY=value` with no quotes, no trailing whitespace, and a single blank line between sections; and no Lakekeeper service-account passwords appear because M2's design disables auth. ^p003

Two template-specific decisions that deviate from a pure-`CHANGEME` policy: `OPENMETADATA_ADMIN_EMAIL` defaults to `admin@atmosphere.local` rather than `CHANGEME` because the field must parse as an email even in the template form, and any operator shipping to production sees it and replaces it just like the surrounding passwords. `GHPAGES_COMMIT_AUTHOR_NAME` defaults to the real value `atmosphere-bot` because the design fixes that name across both the Elementary and Quarto publishers; the matching `GHPAGES_COMMIT_AUTHOR_EMAIL` stays `CHANGEME` because it is environment-specific. ^p004

Forge verified `.env` is already listed in `.gitignore` (lines 1-2 list `.env` and `.env.local`), so committing `.env.example` without a real `.env` sibling is safe and no leaked-secret vector is introduced by this change. ^p005
